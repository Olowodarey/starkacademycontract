# StarkAcademy

A monorepo for the StarkAcademy project.

## Structure

```
starkacademycontract/
├── contract/   # Cairo smart contracts (Starknet)
└── frontend/   # Frontend application (coming soon)
```

## Contract

The `contract/` folder contains the StarkAcademy Cairo smart contracts built on Starknet.

See [`contract/README.md`](./contract/README.md) for details.

## Frontend

The `frontend/` folder will contain the frontend application.
